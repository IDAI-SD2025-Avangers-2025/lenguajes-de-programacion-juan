Los lenguajes de programacion son una herramienta necesaria para la creacion de programas de computadora

Python es un lenguaje de programacion
de alto nivel el cual se empeña por mantener una
alta legibilidad en su codigo.
Soporta parcialmente la orientacion a objetos.

Python

C++ es un lenguaje de programacion
de alto nivel cuyo proposito incial era añadir
funcionalidad a C agregando orientacion a objetos y programacion estructurada

C++

C es un lenguaje de programacion
el cual esta orientado la implementacion en
sistemas operativos dandole un acceso sin restricciones a los componentes de la PC al programador

C

el cual fue diseñado con un proposito universal
dandole enfasis a la productividad y simplicidad, es un lenguaje interpretado
de alto nivel y dinamico y utiliza "recoleccion de basura"
